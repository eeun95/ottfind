```
./gradlew build

```

### 요구사항 기능 구현
1. 리스트 조회
   1. OTT(넷플릭스, 왓챠, 티빙 ..) 리스트 조회
   2. 장르별 컨텐츠 리스트 조회
   3. 컨텐츠 정보 조회
2. 정보 입력/수정
   1. OTT 서비스 추가(+웨이브, 디즈니플러스..)/수정
   2. OTT 컨텐츠 추가(+일타스캔들, 주술회전..)/수정(장르 변경..)
   3. 컨텐츠 정보 추가/수정(주/조연 변경..)
3. 정보 삭제
   1. OTT 서비스 삭제
   2. OTT 컨텐츠 삭제
   3. 컨텐츠 정보 삭제
